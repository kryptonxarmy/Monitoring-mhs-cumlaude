
# Monitoring Mahasiswa Cumlaude

Sebuah website yang ditugaskan oleh profi untuk memonitoring mahasiswa yang menginginkan lulus tepat waktu dengan predikat cumlade


## Acknowledgements

 - Nerxt JS - Frontend
 - Express JS = Backend
 - Prisma ORM


## Documentation

- Backend
    npm run dev



## Installation

Install with npm

```bash
---- backend -------
  cd Backend
  npm install
  npm run dev
  
---- frontend -------
  cd frontend
  npm install
  npm run dev
```
